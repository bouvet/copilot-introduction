# Power BI Track — GitHub Copilot Exercises

This folder mirrors exercises 01-05 from the main course, but adapted for Power BI professionals working with semantic models and PBIP-friendly text artifacts.

## How to use this track

1. Work through folders in order (`01` to `05`).
2. Keep Copilot Chat open while editing the files in each exercise.
3. Treat `.tmdl` files as your editable source assets for a PBIP-style workflow.

## Exercises

| # | Folder | Focus |
|---|---|---|
| 01 | `01-first-contact/` | First Copilot interactions using one easy measure |
| 02 | `02-dax-assistant/` | DAX measure authoring with model context |
| 03 | `03-custom-instructions/` | Team-level Copilot behavior for semantic modeling |
| 04 | `04-github-features/` | PR workflow for a DAX bugfix |
| 05 | `05-skills/` | Slash commands and `@workspace` for model assets |

Use these exercises as text-first preparation before or alongside editing measures in Power BI Desktop with a PBIP project.
